# Project Structure

## Top-Level

- `assets/` - figures, diagrams, and other non-code project assets
- `checkpoints/` - model checkpoints and saved weights
- `configs/` - Hydra configuration files
- `dataset/` - raw, processed, cached, and split dataset artifacts
- `demo/` - demo entry points or notebooks
- `docs/` - project documentation
- `experiments/` - experiment logs, notes, and outputs
- `notebooks/` - exploratory notebooks
- `paper/` - paper drafts and publication assets
- `preprocessing/` - preprocessing scripts and pipelines
- `scripts/` - utility scripts for training, evaluation, and tooling
- `src/` - source code
- `tests/` - automated tests

## Source Layout

- `src/evaluator/` - evaluation logic
- `src/losses/` - custom losses
- `src/models/` - model definitions
- `src/modules/` - reusable neural network modules
- `src/trainer/` - training loops and orchestration
- `src/utils/` - shared utilities
- `src/visualization/` - plotting and visualization helpers
